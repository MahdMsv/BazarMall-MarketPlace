🛒 BazarMall Marketplace

A modern and responsive e-commerce landing page / marketplace UI built with HTML, CSS, and JavaScript.
This project focuses on clean UI design, user experience, and front-end development best practices.

🚀 Live Demo

👉 https://mahdmsv.github.io/BazarMall-MarketPlace

📌 Features
Fully responsive design (mobile, tablet, desktop)
Modern UI/UX layout for an online marketplace
Product listing sections
Hero section with call-to-action
Smooth scrolling and interactive elements
Clean and maintainable code structure
Lightweight and fast performance
🛠️ Tech Stack
HTML5
CSS3 (Flexbox / Grid)
JavaScript (Vanilla JS)
Git & GitHub for version control
